# Simple MP3 Downloader

Simple desktop application to download MP3 audio from YouTube videos.

## Getting Started

### Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (latest stable version recommended).

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/simple-mp3-downloader.git
   cd simple-mp3-downloader
   ```

2. Build and run the application:
   ```bash
   cargo run --release
   ```

## Usage

1. Launch the application.
2. Copy a YouTube video URL (e.g., `https://www.youtube.com/watch?v=...`).
3. Paste the URL into the input field.
4. Click the **Download MP3** button.
5. Select the destination folder and filename in the save dialog.
6. Wait for the download to complete.

## Development

To run tests:
```bash
cargo test
```
